Samir Jihadi - AD Home Lab

Active Directory (AD) is a directory service that is designed to manage and store information about network resources, such as computers, users, and applications, in a centralized and organized manner. Running on VMware, with Windows Server 2022, a full installation of SAM.lab was completed and acted as the Domain Controller. Installations of Windows 10 Enterprise and Kali Linux were also completed alongside the Domain Controller installation and were given static IP addresses as well. A network map diagram and completed pictures of the setup can be found below. An AS-REP roasting attack was also executed against the Domain Controller using Rubeus, where user passwords has Kerebos Pre-authentication disabled, allowing the hashes of the passwords to be stolen.
